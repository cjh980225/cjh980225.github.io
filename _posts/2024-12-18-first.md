# 오늘 처음 블로그를 만들었어요 

앞으로 열심히 해보겠습니다.

```python
n_list = []
not_in_list = []

for i in range(1, 29) : 
    x = int(input(""))
    n_list.append(x)

for i in range(1, 31) : 
    if i not in n_list : 
        not_in_list.append(i)
    else : 
        pass
        
not_in_list.sort()

print(not_in_list[0])
print(not_in_list[1])
```
